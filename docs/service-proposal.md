## 摘要

本計畫針對**囤積物品的長者**，在公共屋邨及社區推行「主題交換日 + 修繕工作坊 + 積分鼓勵 + 多渠道活動提示」整合服務。透過**上門關懷、義工代辦及低壓力漸進釋出**，讓閒置物品在街坊間重新流轉，修繕仍可用的物件，並強化社會連結。一期同步推出**簡易數碼 MVP**（中心後台 + 可選 PWA）支援紀錄與配對。

---

## 1. 背景與理念

### 1.1 社會需要

香港長者住屋面積普遍細小，物品囤積易影響出入動線及屋邨消防安全。不少長者抱持「修好還能用」「總有一天會用到」的心態，難以棄置物品；同時缺乏修繕技能，亦未必能使用一般二手應用程式。若僅以「執屋」「清理」為訊息，易引發羞恥與抗拒。

### 1.2 服務理念

本計畫採 **「保留價值感 + 街坊連結 + 低壓力釋出」** 路徑：

- 物品以**分享、交換、轉贈**呈現，而非垃圾處理  
- **修繕先行**，尊重惜物；交換為後續選項  
- **線下為主、數碼為輔**；義工代登記為常態  
- 單次釋出 1～3 件即可參與，不設「清完才能參加」  

### 1.3 與循環經濟及社會目標的關係

- 延長物品壽命、減少棄置  
- 促進社區互助與長者社會參與  
- 支援數碼包容（無需強制使用智能手機）

## 2. 問題與服務創新

| 現況問題 | 本服務創新 |
|----------|------------|
| 長者缺乏修繕技能，物品閒置 | 修繕師傅上門／定點服務，配對社區技工與義工 |
| 缺乏安全、信任的釋出渠道 | 固定據點、熟識職員、主題交換日 |
| 數碼障礙 | 紙本積分卡、電話與上門代辦 |
| 街坊互動少 | 交換日營造定期聚會，積分鼓勵重複參與 |
| 怕被打標「囤積」 | 去標籤文案、暫存待領、試水溫參與 |

### 2.1 本地服務參考與定位

香港已有相類服務，本計畫採**互補而非重複**策略（詳見 [service-references-hk.md](service-references-hk.md)）：

| 參考服務 | 重點 | 與本計畫關係 |
|----------|------|----------------|
| [Don't Throw Me](https://dontthrowme.com/zh-Hant/service/swap) | 網上 DTM 點數換物（可兌現金） | 本計畫走**線下長者、非儲值積分**，不以平台定價估價 |
| [屯門拍檔・舊物漂流](https://www.threads.com/@tum.pakdong/post/DYmqg7Wk9lU) | 地區舊物交換活動 | 同類「主題交換日」；可加長者上門、修繕、積分紀錄 |
| [哈爾移動椅子](https://www.instagram.com/p/DXRpAlgDVgC/)（howlsmovingchair） | 社區取材、椅子翻新、「一椅一達人」 | 參考升級改造攤；本計畫補**電話／上門**觸及長者 |
| [復修辦館](https://www.instagram.com/repaircafe_hk/) | Repair Café 即場修繕 | 可聯辦修繕攤；本計畫加**上門**及修後交換銜接 |
| [家居維修學院](https://www.repairfairyhk.com/repairCourse/repairCourse.html) | 家居維修課程與義工 | 潛在**師傅／義工培訓**伙伴；高危工程須專業承判 |

> 香港已有網上換物與修繕活動，但針對**囤積物品長者**、結合**上門代辦、非金融積分、修繕—交換銜接**的系統化屋邨服務仍屬缺口。

## 3. 服務內容

### 3.1 主題交換日（每月）

- 按類別舉辦：衣物、餐具、小型家電、書報、孫輩用品等  
- 設於屋邨社區會堂／長者中心；無障礙動線  
- 設交換、修繕、升級改造攤位  
- 「帶多少算多少」；首次可僅參觀；設**暫存待領**  

### 3.2 活動推送

渠道優先序：**電話／上門 > 屋邨告示 > WhatsApp > 短訊**  
內容：本月主題、地點、可只帶一件。長者可選「僅電話聯絡」。

### 3.3 積分錢包

- 名稱：**社區感謝積分**（非儲值、不可兌現金）  
- 用途：修繕優先預約、實用品、茶點  
- 紙本卡與系統並行  

### 3.4 修繕物品工作坊

- 義工協助登記；**以上門修繕為主**（雙人義工）  
- 完成後由長者決定留用或於交換日釋出  
- 無法修：提供合法回收資訊（如環保署 GREEN@COMMUNITY）

## 4. 目標族群

| 類別 | 說明 |
|------|------|
| **主要** | 65 歲以上、有物品囤積傾向的長者（含獨居、行動不便） |
| **支援** | 社區義工、中心職員、家人（經同意）、修繕師傅／技工 |
| **間接** | 同一屋邨街坊（接收交換物品） |

**不包含**：本計畫不提供強制清屋、醫療診斷或心理治療；嚴重個案轉介社署／醫管局社康。

## 7. 預期成效與 KPI

### 7.1 成效指標

| 指標 | 定義 | 首年目標（示例，待調整） |
|------|------|--------------------------|
| 長者參與人次 | elderly_flag 報到數 | ≥ 300 |
| 人均釋出件數 | 每次活動釋出總數／參與人次 | 逐季上升 10% |
| 重複參與率 | 3 個月內 ≥2 次活動之長者占比 | ≥ 40% |
| 修繕完成宗數 | status=completed | ≥ 80 |
| 修繕後釋出比 | 完成修繕後進入交換之比例 | 記錄基線 |
| 義工代操作占比 | ProxyRegistration／總登記 | ≥ 60% |
| 自評社會連結 | 活動後 3 題紙本量表（可選） | 平均分提升 |

### 7.2 質化成效

- 長者自述「唔使一次過執晒」的安心感  
- 街坊關係改善個案故事（去識別化）  
- 中心職員反映轉介個案減少緊急清屋壓力

## 8. 風險與配套

### 8.1 個人資料（PDPO）

- 收集最少資料；書面／口頭同意；代登記留痕  
- 私隱政策公開；支援查閱及更正  

### 8.2 消防安全

- 服務目標為漸進疏導，不替代物管執法  
- 與物管溝通活動性質；提醒通道勿阻塞  

### 8.3 上門服務安全

- 雙人義工；預約；家人知情  
- 電器、氣體相關修繕禁制清單  
- 機構公共責任保險  

### 8.4 轉介機制

懷疑嚴重囤積伴隨自我照顧困難 → 轉介社署／醫管局社康；平台不提供診斷。

### 8.5 積分合規

積分不可兌現金，避免構成儲值支付工具。
