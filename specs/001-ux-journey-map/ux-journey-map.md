## 一、總覽

### 1.1 服務定位

社區換物 Carousell 以**每月換物市集**為核心，結合**線上 App 媒合**與**社區感謝積分**，服務香港街坊（優先關注長者）的物資循環與鄰里連結需求。

**體驗設計原則**：
- 線上媒合、現場交收（ADR-0001）
- 實體為主：唔識用 App 亦有義工幫手
- 低門檻：Walk-in、試水溫參觀均可
- 長者友善：大字、暖色、大觸控區

### 1.2 旅程階段

```mermaid
    S1[1 認知] --> S2[2 考量]

```
